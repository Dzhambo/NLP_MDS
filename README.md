# NLP_MDS

# About

- This is an introductory NLP course for MDS @ HSE
- Lecture and seminar materials for each week are in ./week* folders, see README.md for materials and instructions
- Any technical issues, ideas, bugs in course materials, contribution ideas - add an issue

# Syllabus

- week01 Word embeddings
  - Lecture: Word embeddings
  - Seminar: Word2Vec and Embeddings representation analysis

- week02 Sequence handling, text classification
  - Lecture: Sequence handling, text classification
  - Seminar: Sequence handling for salary prediction

- week03 Language modeling, Seq2seq
  - Lecture: Language modeling, Seq2seq
  - Seminar: Machine translation

- week04 Transformer
  - Lecture: Transformer
  - Seminar: Implementation GPT2

- week05 Transformer tricks & PEFT
  - Lecture: Transformer tricks & PEFT
  - Seminar: PEFT

- week06 LLMs & RLHF
  - Lecture: LLMs, Prompting, RLHF
  - Seminar: RLHF
